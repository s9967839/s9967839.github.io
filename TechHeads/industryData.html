Industry Data


Desmond-Lead Front End Developer

According to Job Market Data (2022), Lead Front End Developer is ranked 6th in the top IT job titles. It is one of the high in-demand roles worldwide (Everything You Need to Know About Front-End Developer Salary | University of the People, 2022). A Lead Front End Developer is responsible for leading the team in building the web interface that people will interact with. The skills required for a Lead Front End Developer are HTML, CSS, and JavaScript. Besides, other IT-specific skills include DOM Manipulation, JSON, AJAX, Node.js, MySQL, React, and Java (11 Front End Developer Skills You Need to Get Hired in 2022, 2022). The Burning Glass Data (2018) ranked JavaScript as the 2nd and Java as the 3rd greatest in-demand IT skills. Communication, active listening, teamwork, leadership, analytical, and problem-solving abilities are among the other requirements for being a Lead Front End Developer and among the top generic skills in highest demand from employers.

The Burning Glass Data (2018) placed Microsoft Windows among the top 3 greatest in-demand IT-specific skills by employers which is not in the required skill set of a Lead Front End Developer. However, the top three generic skills – communication, problem-solving, and organization skills- are this role's requirements.


George- IT Engineer

IT Engineer is ranked in the top 10 list of top in-demand job titles (Job Market Data, 2022). IT Engineer oversees a company's computer systems, including software, hardware, and networks (IT Engineer Job Description - JobHero, 2022). A successful IT Engineer must possess excellent communication and problem-solving skill. Moreover, team player, organization, and interpersonal skills are essential skills required for this role. Comparing the skill sets required for this role with the skills demanded from the employer, communication skills, problem-solving, and organization skills are ranked 1st , 2nd , and 3rd in the Burning Glass Data (2018). The IT-specific skills required for this role are C++ or FORTRAN, Microsoft, VMWare, SQL, and Oracle (IT Engineer Job Description - JobHero, 2022), most of which are listed in the top IT specific skills in greatest demand, with SQL skill ranked at the top.

According to Burning Glass Data (2018), SQL, Java, and JavaScript are among the top three highest-ranked IT-specific skills, among which Java and JavaScript is not included in the required skill set of an IT Engineer. However, the top three generic skills – communication, problem-solving, and organization- are this role's requirements.


Sameh- Chief Technology Officer

Chief Technology Officer is the highest-ranking technology manager in an organization (Medcof, 2008) in charge of managing and maximizing the value of technology within a company (What is a CTO? Your Guide To This Technology Career, 2022). The essential skills required for Chief Technology Officer are communication, leadership, decision-making, organization, business, and technical skills (What is a CTO? Your Guide To This Technology Career, 2022). These skills are also listed as the top skills demanded by employers in Burning Glass Data (2018). The IT-specific skills required for this role are quite broad, including knowledge and experience in software development, product development, programming, system architect, etc. Hence, most of the employers' greatest in-demand IT skills are also required for the role of Chief Technology Officer.

As Chief Technology Officer requires vast generic and IT-specific skills, the skill sets needed for this role meet the employers' list of high in demand skills in Burning Glass data (2018)


Sonam- Application Developer

The Burning Glass Data (2018) placed an Application Developer at 20th place in the top occupation list. The demand for Application Developers is increasing with the growth of smartphones and smartphone apps (IBISWorld - Industry Market Research, Reports, and Statistics, 2022). An Application Developer is a professional who designs, creates, delivers, and updates programs for a specific device, the web, or an operating system (What Is an Application Developer? 2021). An Application developer works with the project management teams and ensures that the client's needs are met (What Is an Application Developer?, 2021). Strong communication skills, problem-solving, attention to detail, and analytical thinking skills are necessary to become an Application Developer (Application Developer Job Description | LinkedIn Talent Solutions, 2021). These generic skills are ranked 1st, 2nd, 8th, and 17th in the top generic skills demanded by employers. In addition, knowledge of programming languages like C++ and Java is essential for this role. While Java is ranked 3rd in the list of greatest demand by employers, C++ is not on the list.

The three highest-ranked IT skills not in the required skill set for this role are Javascript, SQL, and Microsoft Windows. Besides, organization skills, one of the top three highest skills in demand, is not mentioned for an Application Developer.

After looking at the Burning Glass data, my opinion for the ideal job remains the same as this role is something I am passionate about right now. I want to explore the new possibilities as an Application Developer and at the same time keeping the door open for new opportunities.
